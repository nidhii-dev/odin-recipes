# Odin Recipes

A simple recipe website built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

## Description

This project is a basic HTML website that contains a collection of recipes. It was built to practice fundamental HTML skills like links, lists, and page structure.

## What I Learned

- HTML document structure
- Working with links and images
- Using ordered and unordered lists
- Organizing a multi-page website
